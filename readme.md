# 临安浮生记

一款以南宋临安（今杭州）为背景的古风商业模拟游戏。

## 游戏背景

公元1140年，岳飞北伐抗金之际，你作为一名初到临安的商人，背负着5500文的债务。你必须在40天内通过倒买倒卖积累财富，还清债务并成为临安城首富。

## 游戏特色

- 🏛️ 南宋古风背景，沉浸式历史体验
- 💰 真实的商业交易系统
- 🎲 随机事件增加游戏挑战性
- 🎵 古风背景音乐营造氛围
- 📱 响应式设计，支持多设备

## 游戏玩法

1. **地点选择**: 在临安城的不同地区间移动
2. **商品交易**: 买入低价商品，在其他地区高价卖出
3. **风险管理**: 灰色商品利润高但风险大
4. **时间管理**: 合理安排40天的行程
5. **资金周转**: 及时还债，避免利息累积

## 技术特性

- 纯HTML/CSS/JavaScript实现
- 本地存储游戏状态
- 响应式设计
- 音频管理系统
- 动画效果

## 开始游戏

直接打开 `index.html` 文件即可开始游戏。

## 游戏目标

在40天内：
- 还清5500文债务
- 积累更多财富
- 成为临安城首富

## 注意事项

- 每天只能访问一个地点
- 灰色商品有被查获的风险
- 合理管理健康值
- 注意市场价格波动
